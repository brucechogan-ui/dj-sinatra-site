# DJ Sinatra — Official Website

A minimal, luxury-aesthetic static website for DJ Sinatra.

## Stack

- HTML, CSS, vanilla JavaScript
- No frameworks, no build step
- Google Fonts (Cormorant Garamond + Inter)

## Deploy

Push to GitHub and connect to **Vercel** — it will deploy automatically with zero configuration.

Or run locally:

```bash
npx serve .
```

## Structure

```
index.html   — Single-page site
styles.css   — All styles (mobile-first)
script.js    — Scroll reveal + parallax
README.md    — This file
```

## Customize

- Replace `INSERT_HANDLE` in `index.html` with the Instagram handle
- Replace `booking@domain.com` with the real booking email
- Swap Google Fonts link if different typography is preferred
